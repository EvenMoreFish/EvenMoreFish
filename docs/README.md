# EMF Documentation & Website

Built with [Docusaurus](https://docusaurus.io/).

## Quick Start

1. Install dependencies:
   ```bash
   yarn
   ```
   or
   ```bash
   npm install
   ```

2. Run locally:
   ```bash
   yarn start
   ```
   or
   ```bash
   npm run start
   ```

3. Build for production:
   ```bash
   yarn build
   ```
   or
   ```bash
   npm run build
   ```
