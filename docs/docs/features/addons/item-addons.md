---
title: Item Addons
description: Third Party Plugins & Internal addons
---

| Required Plugin | Prefix      | Format                     | Min Java |
|-----------------|-------------|----------------------------|----------|
| ItemsAdder      | itemsadder  | `itemsadder:namespace:id`  | 21       |
| HeadDatabase    | headdb      | `headdb:id`                | 21       |
| Denizen         | denizen     | `denizen:id`               | 21       |
| EcoItems        | ecoitems    | `ecoitems:id`              | 21       |
| Oraxen          | oraxen      | `oraxen:id`                | 21       |
| Nexo            | nexo        | `nexo:id`                  | 21       |
| CraftEngine     | craftengine | `craftengine:namespace:id` | 21       |

Example (ItemsAdder):
```yaml
Salmon:
   item:
     material: "itemsadder:fish:salmon"
```
