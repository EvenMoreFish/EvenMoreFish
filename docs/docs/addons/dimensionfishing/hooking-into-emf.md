---
title: Hooking into EvenMoreFish
sidebar_position: 3
---

DimensionFishing hooks into EvenMoreFish by default. Installing both plugins alongside each other will work as intended.

The default behavior allows any fish to be caught with any type of fishing. Because this may not be what people want, a new RequirementType has been added to the main plugin. See [requirements](https://evenmorefish.github.io/EvenMoreFish/docs/features/requirements#fishing-type) for usage.
