### EMFFishStew Admin Commands

| Command            | Arguments          | Permission         | Description                                        |
|--------------------|--------------------|--------------------|----------------------------------------------------|
| `/fishstew reload` | None               | `fishstew.command` | Reloads plugin config.                             |
| `/fishstew give`   | `<id>`, `<target>` | `fishstew.command` | Gives the selected player(s) the chosen fish stew. |
