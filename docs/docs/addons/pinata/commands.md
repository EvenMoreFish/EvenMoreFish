### EMFPiñata Admin Commands

| Command             | Arguments | Permission          | Description                                   |
|---------------------|-----------|---------------------|-----------------------------------------------|
| `/emfpinata reload` | None      | `emfpinata.command` | Reloads plugin config.                        |
| `/emfpinata pinata` | `<id>`    | `emfpinata.command` | Spawns the specified piñata at your location. |
