## Artifacts

Currently, EMF has 2 main artifacts for use:
- `even-more-fish-api`
- `even-more-fish-plugin`

In the future these will be further abstracted so you don't need to use the "core" artifact any longer.

## CodeMC

The plugin is available via CodeMC's Maven Repository. To add it to your build tool, follow the examples below:

<Tabs>
<TabItem value="gradle" label="Gradle (Kotlin)">

```kotlin title="build.gradle.kts"
repositories {
    maven("https://repo.codemc.io/repository/EvenMoreFish/")
}

// Note: The version shown here may not be the latest. Make sure to check the latest version before using!
dependencies {
    // For the plugin artifact
    // The exclusions are required as your build will fail without them.
    compileOnly("com.oheers.evenmorefish:even-more-fish-plugin:2.1.6") {
        exclude("de.tr7zw", "item-nbt-api")
        exclude("com.github.Anon8281", "UniversalScheduler")
    }
    // For the api artifact
    compileOnly("com.oheers.evenmorefish:even-more-fish-api:2.1.6")
}
```

</TabItem>
<TabItem value="maven" label="Maven">

    ```xml title="pom.xml"
    <repositories>
        <repository>
            <id>evenmorefish</id>
            <url>https://repo.codemc.io/repository/EvenMoreFish</url>
        </repository>
    </repositories>

    <!-- Note: The version shown here may not be the latest. Make sure to check the latest version before using! -->
    <dependencies>
        <!-- For the plugin artifact -->
        <!-- The exclusions are required as your build will fail without them. -->
        <dependency>
            <groupId>com.oheers.evenmorefish</groupId>
            <artifactId>even-more-fish-plugin</artifactId>
            <version>2.0.12</version>
            <scope>provided</scope>
            <classifier>shadow</classifier>
            <exclusions>
                <exclusion>
                    <groupId>de.tr7zw</groupId>
                    <artifactId>item-nbt-api</artifactId>
                </exclusion>
                <exclusion>
                    <groupId>com.github.Anon8281</groupId>
                    <artifactId>UniversalScheduler</artifactId>
                </exclusion>
            </exclusions>
        </dependency>
        <!-- For the api artifact -->
        <dependency>
            <groupId>com.oheers.evenmorefish</groupId>
            <artifactId>even-more-fish-api</artifactId>
            <version>2.0.12</version>
            <scope>provided</scope>
        </dependency>
    </dependencies>
    ```

</TabItem>
